take notes from the commandline and put them in git

Download .zip Download .tar.gz View on GitHub

I allways end with lot's of text files floating around my home directory, so I made a trivial script. Save the comments to a file named as the first argument on a common directory, and make a git commit. If you give the name of the file only, it shows it's contents. If you give nothing to the script, it lists the notes you have.

If you reuse the same name, updates are appended to the file. It could be used in collaboration with other people too.

That's it, have fun.
